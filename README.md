# Todo List Web App

A simple and responsive Todo List application built using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Date reminder
- Data stored in LocalStorage
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open index.html in your browser
